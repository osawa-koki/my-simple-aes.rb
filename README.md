# my-simple-aes.rb

🎭🎭🎭 RubyのAES暗号を使ってみる！  

## 実行方法

```shell
ruby ./app/AES-256-ECB.rb --key "key-key" --data "Hello World!!!"
ruby ./app/AES-256-CBC.rb --iv "initial-vector" --key "key-key" --data "Hello World!!!"
ruby ./app/AES-256-CTR.rb  --iv "initial-vector" --key "key-key" --data "Hello World!!!"
```
